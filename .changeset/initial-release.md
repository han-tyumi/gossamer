Initial release of gossamer — Web API bindings for Gleam.

Includes: Promise, Streams, URL, Headers, Request/Response, fetch,
WebSocket, Crypto, Blob, File, FormData, TextEncoder/Decoder,
Uint8Array, ArrayBuffer, AbortController, JSON, compression, and timers.

Tested across Deno, Node.js, and Bun.
